# Trade Nexus

This repository contains planning docs and initial scaffolding code for **Trade Nexus**, a B2B marketplace connecting global buyers and suppliers.

## Structure
- `docs/PLAN.md` – full product strategy, architecture and roadmap.
- `frontend/` – Next.js + Tailwind starter.
- `backend/` – Express + Prisma starter with PostgreSQL schema.

## Getting Started
```bash
npm install
npm run dev --workspace=frontend
npm run dev --workspace=backend
```

## Testing
Run tests (placeholder):
```bash
npm test --workspaces
```
